Алгоритм кодирования Хэмминга



Предварительно определим последовательности натуральных чисел исходя из их представления в двоичной системе счисления.
  • V1 – 1, 3, 5, 7, 9,... – все числа, у которых разряд № 1 равен 1,
  • V2 – 2, 3, 6, 7, 10,... – все числа, у которых разряд № 2 равен 1,
  • V3– 4, 5, 6, 7, 12,... – все числа, у которых разряд № 3 равен 1,
и т. д. Последовательность Vt начинается с числа 2t-1.


Сообщение A = a1, a2, …, am, слово длины m в двубуквенном алфавите {0,1}, кодируется словом B = b1, b2 , …, bn длины n > m в том же алфавите. Полагаем k = n – m.


Из неравенства 2m ≤ 2n /(n + 1) находим значение n.

В кодовом слове B = b1, b2, …, bn разряды с номерами 20=1, 21 =2, 22=4, ... , 2k-1 называются контрольными, их число равно k. Остальные m разрядов информационные, в них помещаются m букв сообщения a1, a2, …, am.

Контрольные разряды заполняются в соответствии со следующими равенствами:
  • b1 = b3 ⊕ b5 ⊕ b7 ⊕ ... , т. е. сумма всех информационныхразрядов с номерами из V1 (сумма всех, кроме первого),
  • b2 = b3 ⊕ b6 ⊕ b7 ⊕ ... , т. е. сумма всех информационных разрядов с номерами из V2 (сумма всех, кроме первого),
  • b4 = b5 ⊕b6 ⊕ b7 ⊕ ..., т. е. сумма всех информационных разрядов с номерами из V3 (сумма всех, кроме первого),
