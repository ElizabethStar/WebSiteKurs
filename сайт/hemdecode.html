Алгоритм декодирования Хэмминга



Пусть после прохождения через канал связи сообщения b1, b2, … , bn получено сообщение c1, c2, …, cn, отличающееся от него возможно лишь в позиции с номером I = it…i1, где
  • i1 = c1 ⊕ c3 ⊕ c5 ⊕ c7 ⊕ … , т. е. сумма всех разрядов с номерами из V1,
  • i2 = c2 ⊕ c3 ⊕ c6 ⊕ c7 ⊕ …, т. е. сумма всех разрядов с номерами из V2,
  • i3 = c4 ⊕ c5 ⊕ c6 ⊕ c7 ⊕… , т. е. сумма всех разрядов с номерами из V3



Пример: 1100101


  • i1 = c1 ⊕ c3 ⊕ c5 ⊕ c7 = 1 ⊕ 0 ⊕ 1 ⊕ 1 =1,
  • i2 = c2 ⊕ c3 ⊕ c6 ⊕ c7 = 1 ⊕ 0 ⊕ 0 ⊕ 1 = 0,
  • i3 = c4 ⊕ c5 ⊕ c6 ⊕ c7 = 0 ⊕ 1 ⊕ 0 ⊕ 1 = 0.

Получаем, что ошибка произошла в первом разряде I=0012=1;

Исправим ошибку: 0100101

Исходное сообщение: 0101
